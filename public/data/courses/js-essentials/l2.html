DOM Basics

The DOM lets you create interactive pages.

  1. Select elements with document.querySelector
  2. Change text: el.textContent = "Hello"
  3. Listen to events: el.addEventListener("click", ...)
Mini exercise:

Copy this HTML to a local test page, and wire the button:

<button id="btn">Click me</button>
<script>
document.getElementById("btn").addEventListener("click", () => {
  alert("clicked!");
});
</script>
